# COVID-19 Data Visualization Project

![COVID-19](images/covid.png)

This project analyzes and visualizes global COVID-19 data using Python, Pandas, Plotly, and Folium.  
The notebook includes:

## Features
- Scatter Mapbox visualizations of COVID-19 confirmed cases.
- Animated time-series map using Plotly.
- Folium heatmaps and circle markers.
- Country-level statistics: Confirmed, Deaths, Recovered.

## Files Included
- `covid19.ipynb` → Main Google Colab notebook  
- `data/` → CSV files used in analysis  
- `plots/` → Exported charts (if available)
